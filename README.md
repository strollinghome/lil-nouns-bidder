# Lil Nouns Bidder


Install forge
## Install Foundry
See the [Foudry Github](https://github.com/gakonst/foundry#installation) repo for installation.

### Install libraries
```
forge install
```

### Set up remappings
```
forge remappings > remappings.txt
```


## Run
```
forge run ./src/Contract.sol -vvvv --rpc-url ${MAINNET_URL}
```
